---

### **STEP 4: Verify Ownership (HTML Tag Method)**

This is the easiest method for GitHub Pages:

1. Google will show several verification methods. Select **"HTML tag"**

2. Google will provide a meta tag that looks like this:
   ```html
   <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE" />
   ```

3. **Copy this entire line**

4. Open your `index.html` file and add this line in the `<head>` section, right after the existing meta tags (around line 9).

   **Example placement:**
   ```html
   <head>
       <meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
       <meta name="description" content="...">
       <meta name="google-site-verification" content="YOUR_CODE_HERE" />
       ...
   ```

5. Save and commit the file:
   ```bash
   cd ~/git/Indranil2020.github.io
   git add index.html
   git commit -m "Add Google Search Console verification"
   git push origin main
   ```

6. Wait 2-3 minutes for GitHub Pages to update

7. Go back to Google Search Console and click **"Verify"**

8. ‚úÖ You should see **"Ownership verified"**

---

### **STEP 5: Submit Your Sitemap**

After verification:

1. In Google Search Console, look at the left sidebar
2. Click **"Sitemaps"**

   ```
   sitemap.xml
   ```
4. Click **"Submit"**
5. ‚úÖ Status should change to **"Success"** after a few minutes

### **STEP 6: Request Indexing**

This tells Google to crawl your site immediately:

1. In Google Search Console left sidebar, click **"URL Inspection"**


4. Google will check the URL
5. Click **"Request Indexing"**
6. Wait 1-2 minutes while Google queues your request
7. ‚úÖ You'll see **"Indexing requested"**

## üÜò TROUBLESHOOTING

### Problem: "Verification Failed"
**Solution:**
- Double-check the verification code in index.html
- Make sure you committed and pushed to GitHub
- Wait 5 minutes for GitHub Pages to update
- Clear your browser cache
- Try verification again

### Problem: "Sitemap couldn't be read"
**Solution:**
- Check the sitemap URL: `https://indranil2020.github.io/sitemap.xml`
- Open it in browser to verify it loads
- Wait 24 hours and try submitting again

### Problem: "Site not appearing in search after 2 weeks"
**Solution:**
- Check Google Search Console for errors
- Verify sitemap is submitted and processed
- Request indexing again
- Build more backlinks
- Ensure robots.txt isn't blocking search engines

### Problem: "Ranking is low"
**Solution:**
- Build more high-quality backlinks
- Add more content to your page
- Update content regularly
- Get links from .edu and .gov sites
- Share on academic social networks
