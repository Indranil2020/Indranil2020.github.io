About This Tool

The Basis Set Visualizer is an interactive Streamlit application that allows researchers and students to explore and visualize atomic basis sets used in Density Functional Theory (DFT) calculations. Access basis sets from the comprehensive Basis Set Exchange database with real-time 3D visualization.

Features

  • Browse and search the Basis Set Exchange (BSE) database
  • Interactive 3D visualization of basis functions
  • Real-time parameter adjustments and exploration
  • Export basis set data for use in DFT calculations
  • Support for various elements and basis set families
  • Educational tool for understanding basis set construction
  • Compare different basis sets side-by-side
  • Free and open-source

Perfect for:

Students learning quantum chemistry, researchers selecting basis sets for calculations, educators teaching computational chemistry, and anyone interested in understanding the mathematical foundations of DFT calculations.

Deployment Instructions

To deploy this Streamlit app, follow these steps:

  1. Deploy to Streamlit Community Cloud (free hosting):
    • Go to streamlit.io/cloud
    • Connect your GitHub repository
    • Select the DFT_visual repository
    • Deploy and get a public URL
  2. Or deploy to Render, Heroku, or Railway
  3. Update the link in this page to point to your deployed app

Technical Details

Technology Stack:

  • Python with Streamlit framework
  • Basis Set Exchange (BSE) Python library
  • Plotly for 3D visualization
  • NumPy and SciPy for numerical computations
  • Pandas for data handling
