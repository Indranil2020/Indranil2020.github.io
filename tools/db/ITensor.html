Overview

ITensor is a powerful C++ library for tensor network calculations with a focus on ease of use through intelligent index matching. Developed by Miles Stoudenmire and collaborators, ITensor provides sophisticated tools for DMRG (Density Matrix Renormalization Group), MPS (Matrix Product States), and general tensor network algorithms. The library emphasizes physicist-friendly notation where indices automatically contract when matched, making complex tensor network code more readable and less error-
