Overview

TeNPy is a comprehensive Python library for tensor network algorithms, particularly focused on one-dimensional quantum systems and DMRG (Density Matrix Renormalization Group). Developed with emphasis on accessibility and ease of use, TeNPy provides a pure Python implementation of state-of-the-art tensor network methods with excellent documentation and a clean, modular design.
