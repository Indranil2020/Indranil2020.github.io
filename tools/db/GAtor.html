Overview

## Capabilities (CRITICAL) - See official documentation for complete capabilities list GAtor is a massively parallel, first-principles genetic algorithm (GA) for molecular crystal structure prediction. Written in Python, it interfaces with the FHI-aims code for local optimizations and energy evaluations using dispersion-inclusive DFT.
