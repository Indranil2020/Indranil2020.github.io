Capabilities (CRITICAL)

  • See official documentation for complete capabilities list Status: ℹ️ MODULE

"PyXtal-ML" refers to the machine learning features embedded within the PyXtal library. It is not a separate package. PyXtal includes tools for:

  • Generating crystal structures with symmetry constraints for ML training sets.
  • Calculating symmetry-weighted structural descriptors.
  • Interfacing with machine learning potentials for structure relaxation.
