Overview

JDFTx (Joint Density Functional Theory - extended) is a plane-wave DFT code designed for electronic structure calculations in molecules, surfaces, and liquids, with particular emphasis on solvation and electrochemistry. Developed by Ravishankar Sundararaman and collaborators, JDFTx features efficient joint density functional theory for implicit solvation, advanced models for electrochemical interfaces, and GPU acceleration.
