Overview

ParAutomatik is a cutting-edge workflow automation tool designed for the parameterization of Density Functional Tight Binding (DFTB) models using Machine Learning. It addresses one of the biggest bottlenecks in semi-empirical methods: the difficulty of creating accurate parameters.
