Overview

**WOPTIC** is a code designed to calculate frequency-dependent optical conductivity ($\sigma(\omega)$) and related transport properties using **Maximally Localized Wannier Functions (MLWFs)**. Its distinctive feature is the use of an **adaptive k-mesh refinement scheme** based on the tetrahedron method, which allows it to efficiently resolve fine spectral features (such as those arising from band crossings or van Hove singularities) that would require prohibitively dense uniform grids.
