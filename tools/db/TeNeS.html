Overview

TeNeS (Tensor Network Solver) is an open-source, massively parallel software package for calculating the ground-state wavefunctions of two-dimensional quantum many-body systems. It employs the infinite Projected Entangled Pair States (iPEPS) ansatz and optimizes it using imaginary time evolution.
