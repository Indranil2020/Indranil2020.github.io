## Capabilities (CRITICAL) - See official documentation for complete capabilities list SyMat (Symmetry-aware generation of periodic Materials) is a generative model that incorporates crystal symmetry into the generation process. It uses a symmetry-aware probabilistic model in the coordinate diffusion process.
