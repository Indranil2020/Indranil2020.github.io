Overview

## Capabilities (CRITICAL) - See official documentation for complete capabilities list CSPML (Crystal Structure Prediction with Machine Learning) is a template-based crystal structure prediction method using metric learning for element substitution. It predicts stable structures by selecting templates from known crystal structures.
