## Overview
teMatDb is a thermoelectric materials database project aimed at consolidating thermoelectric property data extracted from the literature, with accompanying utilities for analysis and handling of the dataset.

**Scientific domain**: Thermoelectric materials data, transport properties curation  
**Target user community**: Researchers performing data-driven thermoelectric screening/analysis
