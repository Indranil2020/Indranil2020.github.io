## Overview
The Nanocrystallography Database (NCD) is a collection of crystal structure data, primarily focused on nanomaterials, minerals, and zeolites. It is closely affiliated with the Crystallography Open Database (COD) and provides an open-access platform for identifying and retrieving crystallographic information (CIF files).

**Scientific domain**: Crystallography, database, nanomaterials
**Target user community**: Crystallographers, mineralogists, materials scientists

## Capabilities
- **Database**: Searchable collection of crystal structures.
- **Focus**: Nanocrystalline materials and minerals.
- **Open Access**: Freely available data.
- **Visualization**: Integrated 3D structure visualization.

## Interfaces & Ecosystem
- **COD**: Part of the Crystallography Open Database ecosystem.
- **CIF**: Uses the standard Crystallographic Information File format.

## Workflow and Usage
1. Access the web interface.
2. Search for materials by composition, unit cell, or name.
3. Download CIF files for use in simulation software.
