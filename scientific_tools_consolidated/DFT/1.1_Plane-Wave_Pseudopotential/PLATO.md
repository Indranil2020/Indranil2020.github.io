## Overview
PLATO is a Chinese-developed first-principles calculation package using numerical atomic orbitals, designed for large-scale electronic structure calculations. Developed primarily in China with integration into ABACUS ecosystem, PLATO implements efficient algorithms for DFT calculations with emphasis on numerical orbitals and large systems. It represents part of China's efforts in developing indigenous computational materials science tools.

**Scientific domain**: Numerical atomic orbitals, DFT, large-scale calculations  
**Target user community**: Materials scientists, Chinese research community, large system researchers
