## Overview
JuKKR is a modern KKR (Korringa-Kohn-Rostoker) Green's function DFT code developed at Forschungszentrum Jülich, Germany. Part of the JuDFT family of codes, JuKKR provides KKR method capabilities with Python interfaces (masci-tools, aiida-kkr) for high-throughput workflows and materials informatics. It implements multiple scattering theory for electronic structure, spectroscopy, and disordered alloys.

**Scientific domain**: KKR Green's function DFT, alloys, spectroscopy  
**Target user community**: Materials scientists, alloy researchers, workflow developers
