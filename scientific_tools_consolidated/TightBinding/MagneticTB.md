## Overview
MagneticTB is a Python package for tight-binding calculations of magnetic systems, with special focus on magnetic topological materials and spin-dependent phenomena. The code provides tools for constructing magnetic tight-binding models, calculating spin-dependent band structures, and analyzing magnetic topological properties.

**Scientific domain**: Magnetic tight-binding, magnetic topological materials, spintronics  
**Target user community**: Magnetic materials, topological magnetism, spintronics researchers
