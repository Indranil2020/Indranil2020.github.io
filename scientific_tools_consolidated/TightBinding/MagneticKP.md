## Overview
MagneticKP is a Python package for k·p theory calculations in magnetic systems. The code provides tools for constructing and solving k·p Hamiltonians for magnetic materials, enabling analysis of band structures, effective models, and low-energy physics near band extrema. MagneticKP complements tight-binding approaches with effective k·p descriptions for magnetic systems.
