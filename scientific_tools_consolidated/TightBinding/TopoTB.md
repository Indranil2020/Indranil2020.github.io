## Overview
TopoTB is a Python package for constructing and analyzing tight-binding models with emphasis on topological properties. The code provides tools for building TB Hamiltonians, calculating topological invariants, and analyzing edge/surface states. TopoTB aims to simplify topological analysis workflows with a Python-friendly interface.
