## Overview
TopoTB is a software package written in Mathematica for calculating electronic structures, topological properties, and phase diagrams of tight-binding models. Developed by Xielin Huang and collaborators, it features a user-friendly interactive interface that allows for real-time tuning of model parameters and visualization of results. It supports calculations of Berry curvature, Chern numbers, Z2 invariants, and surface states.

**Scientific domain**: Tight-binding, Topological Physics, Phase Diagrams, Mathematica
**Target user community**: Condensed matter theorists, Topological materials researchers

## Theoretical Methods
- Tight-Binding Hamiltonians
- Berry Curvature & Berry Phase
- Topological Invariants (Chern, Z2)
- Wilson Loop method
- Surface/Edge state calculations
- Phase diagram mapping
