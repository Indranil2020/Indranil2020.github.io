## Overview
ir2tb is a Python tool for constructing symmetry-adapted tight-binding models from irreducible representations. Developed by Zhijun Wang, ir2tb automates the construction of tight-binding Hamiltonians that respect crystal symmetries by utilizing group theory and irreducible representations. The code simplifies building symmetry-correct TB models for research and analysis.
