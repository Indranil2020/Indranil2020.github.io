# vaspkit

## 1. Description
**vaspkit** is a command-line tool for pre- and post-processing VASP calculations. It provides a user-friendly interface to generate input files and analyze output data.

## 2. Capability Verification
- **Functionality**:
  - Pre-processing: Input file generation (K-points, band paths)
  - Post-processing: Band structure, DOS, Fermi surface, optical properties, charge density analysis
- **Key Features**:
  - Automatic K-path generation
  - 2D/3D visualization data generation
  - Elastic constants calculation
  - Carrier mobility estimation
