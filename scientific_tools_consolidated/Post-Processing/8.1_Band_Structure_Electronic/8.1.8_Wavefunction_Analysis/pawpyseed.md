# pawpyseed

## Overview
**pawpyseed** is a parallel C/Python package for numerical analysis of PAW (Projector Augmented Wave) DFT wavefunctions from VASP calculations.

## Official Resources
- **GitHub**: https://github.com/kylebystrom/pawpyseed
- **PyPI**: https://pypi.org/project/pawpyseed/

## Capabilities
- **Wavefunction Reconstruction**: Full all-electron wavefunctions
- **PAW Analysis**: Projector function handling
- **Overlap Calculations**: Wavefunction overlaps
- **Defect Analysis**: Defect wavefunction characterization

## Key Features
- WAVECAR parsing
- POTCAR projector reading
- All-electron wavefunction reconstruction
- Parallel processing (C backend)
- Python interface

## Applications
- Defect calculations
- Wavefunction analysis
- Charge density studies
- Core-level analysis

## Installation
```bash
pip install pawpyseed
```
