# Brillouin-zone-navigator

## Overview
**Brillouin-zone-navigator** is an interactive tool for visualizing and navigating Brillouin zones of crystal structures.

## Official Resources
- **GitHub**: Search for Brillouin zone navigator

## Capabilities
- **BZ Visualization**: 3D Brillouin zone display
- **High-symmetry Points**: Identify special k-points
- **K-path Selection**: Interactive path creation
- **Crystal Systems**: Support for all Bravais lattices

## Key Features
- Interactive 3D visualization
- High-symmetry point labeling
- K-path generation
- Educational tool

## Applications
- Band structure path planning
- Teaching crystallography
- K-space navigation
