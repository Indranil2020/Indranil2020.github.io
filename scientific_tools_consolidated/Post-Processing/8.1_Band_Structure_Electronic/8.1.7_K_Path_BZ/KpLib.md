# KpLib

## Overview
**KpLib** is a k-point grid generation library providing optimal k-point meshes for DFT calculations based on the Mueller group's research.

## Official Resources
- **GitLab**: https://gitlab.com/muellergroup/kplib
- **Web Server**: https://muellergroup.jhu.edu/K-Points.html

## Capabilities
- **Optimal K-grids**: Generate efficient k-point meshes
- **Symmetry-aware**: Account for crystal symmetry
- **Convergence**: Ensure calculation convergence

## Key Features
- C++ API
- Python wrapper
- Web interface available
- Customizable parameters

## Theory
Based on research for generating optimal k-point grids that minimize computational cost while maintaining accuracy.
