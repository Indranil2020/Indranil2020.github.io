# UltimateEPM

## Overview
**UltimateEPM** is an implementation of the Empirical Pseudopotential Method (EPM) for calculating electronic band structures of semiconductors.

## Official Resources
- **GitHub**: Search for UltimateEPM empirical pseudopotential

## Capabilities
- **EPM Calculations**: Empirical pseudopotential method
- **Band Structure**: Semiconductor bands
- **Form Factors**: Pseudopotential parameterization
- **Alloys**: Virtual crystal approximation

## Key Features
- Classic EPM implementation
- Semiconductor focus
- Fast calculations
- Educational tool

## Theory
The EPM uses empirical form factors fitted to experimental data to calculate band structures without full DFT calculations.

## Applications
- Semiconductor band structures
- Alloy band engineering
- Educational purposes
- Quick band estimates
