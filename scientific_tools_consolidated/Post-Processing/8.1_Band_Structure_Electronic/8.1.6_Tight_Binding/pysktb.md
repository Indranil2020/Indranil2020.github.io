# pysktb

## Overview
**pysktb** is a Python package for Slater-Koster tight-binding calculations with a focus on topological materials analysis.

## Official Resources
- **GitHub**: Search for pysktb Slater-Koster

## Capabilities
- **Slater-Koster TB**: Two-center approximation
- **Topological Analysis**: Berry phase, Chern numbers
- **Band Structure**: Electronic band calculations
- **Spin-Orbit Coupling**: SOC implementation

## Key Features
- Slater-Koster parameterization
- Topological invariant calculation
- Spin-orbit coupling
- Python interface

## Applications
- Topological insulators
- Weyl/Dirac semimetals
- 2D materials
- Heterostructures
