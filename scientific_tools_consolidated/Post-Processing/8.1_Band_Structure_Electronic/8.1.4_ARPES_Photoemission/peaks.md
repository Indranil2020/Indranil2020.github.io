# peaks

## Overview
**peaks** is a modern Python framework for ARPES (Angle-Resolved Photoemission Spectroscopy) data analysis. It provides comprehensive tools for processing and analyzing photoemission data.

## Official Resources
- **GitHub**: Search for peaks ARPES Python

## Capabilities
- **Data Loading**: Multiple file format support
- **Calibration**: Energy and angle calibration
- **Band Analysis**: EDC/MDC fitting
- **Visualization**: Interactive plotting

## Key Features
- Modern Python implementation
- Interactive analysis
- Multiple data formats
- Publication-ready figures

## Applications
- ARPES data analysis
- Band structure extraction
- Fermi surface mapping
