# mpes

## Overview
**mpes** is a Python toolkit for multidimensional photoemission spectroscopy data analysis. It provides tools for processing and analyzing time-resolved and angle-resolved photoemission data.

## Official Resources
- **GitHub**: Search for mpes photoemission

## Capabilities
- **Data Processing**: ARPES/trARPES data handling
- **Calibration**: Energy and momentum calibration
- **Visualization**: Multi-dimensional data plotting
- **Analysis**: Band structure extraction

## Key Features
- Multidimensional data support
- Time-resolved ARPES (trARPES)
- Calibration routines
- HDF5 data handling

## Applications
- ARPES experiments
- Time-resolved photoemission
- Ultrafast dynamics studies
