# ARPESGUI

## Overview
**ARPESGUI** is a MATLAB-based graphical user interface for analyzing soft X-ray angle-resolved photoemission spectroscopy (SX-ARPES) data.

## Official Resources
- **Platform**: MATLAB
- **Application**: SX-ARPES data analysis

## Capabilities
- **Data Loading**: Import ARPES measurement data
- **Band Mapping**: Extract electronic band structures
- **Energy Distribution**: EDC/MDC analysis
- **Fermi Surface**: k-space mapping

## Key Features
- Graphical user interface
- Interactive data exploration
- Band structure extraction
- Publication-ready figures

## Inputs & Outputs
- **Inputs**: ARPES measurement data files
- **Outputs**: Band structures, Fermi surfaces, EDC/MDC curves

## Limitations & Known Constraints
- **MATLAB required**: Needs MATLAB license
- **SX-ARPES focus**: Specialized for soft X-ray ARPES
- **GUI-based**: Less scriptable than Python alternatives

## Comparison with Other Tools
- **vs PyARPES**: ARPESGUI MATLAB, PyARPES Python
- **vs mpes**: Different feature sets and platforms
- **Unique strength**: GUI for soft X-ray ARPES analysis
