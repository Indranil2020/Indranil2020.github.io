# effmass

## Overview
**effmass** is a Python package for calculating effective masses from DFT band structure calculations. It provides tools for extracting carrier effective masses from electronic band structures.

## Official Resources
- **GitHub**: https://github.com/lucydot/effmass
- **Documentation**: https://effmass.readthedocs.io/
- **PyPI**: https://pypi.org/project/effmass/

## Capabilities
- **Effective Mass Calculation**: From DFT bands
- **Parabolic Fitting**: Band curvature analysis
- **Non-parabolic Bands**: Kane model support
- **Multi-valley**: Handle multiple band extrema

## Key Features
- VASP support
- Automatic extrema detection
- Parabolic and non-parabolic models
- Optical effective mass
- DOS effective mass

## Supported Codes
- VASP (vasprun.xml)
- FHI-aims
- Castep

## Installation
```bash
pip install effmass
```
