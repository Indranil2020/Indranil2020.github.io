# Effective-mass-fitting

## Overview
**Effective-mass-fitting** is a PyQt-based application for parabolic band fitting to extract effective masses from DFT calculations.

## Official Resources
- **GitHub**: Search for effective mass fitting PyQt

## Capabilities
- **Parabolic Fitting**: Fit bands to parabola
- **Interactive GUI**: PyQt interface
- **Visual Fitting**: Graphical band selection
- **Mass Extraction**: Calculate effective masses

## Key Features
- Graphical user interface
- Interactive band selection
- Real-time fitting
- Export results

## Applications
- Effective mass calculation
- Band curvature analysis
- Semiconductor characterization
