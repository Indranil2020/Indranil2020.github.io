# vaspvis

## Overview
**vaspvis** is a highly flexible and customizable Python library for visualizing electronic structure data from VASP calculations. It provides an easy-to-understand method for generating a wide variety of band structure and density of states plots.

## Official Resources
- **GitHub**: https://github.com/DerekDardzinski/vaspvis
- **Documentation**: https://derekdardzinski.github.io/vaspvis/
- **PyPI**: https://pypi.org/project/vaspvis/

## Capabilities
- **Band Structure Plotting**: Standard, projected, and spin-polarized band structures
- **DOS/PDOS**: Total and projected density of states
- **Orbital Projections**: Atom and orbital resolved plots
- **Spin Textures**: Spin-polarized visualization
- **Customization**: Extensive matplotlib-based customization options

## Key Features
- Parse EIGENVAL, PROCAR, KPOINTS, POSCAR, INCAR files
- Support for spin up/down channels
- Atom/orbital/element projections
- High-quality publication-ready figures
- Easy integration with matplotlib

## Inputs & Outputs
- **Inputs**: VASP output files (EIGENVAL, PROCAR, KPOINTS, POSCAR, INCAR)
- **Outputs**: Matplotlib figures, PNG/PDF/SVG plots
