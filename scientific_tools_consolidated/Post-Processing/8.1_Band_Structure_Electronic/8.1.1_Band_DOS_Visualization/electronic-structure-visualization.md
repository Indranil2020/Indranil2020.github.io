## Overview
electronic-structure-visualization is an interactive Dash/Plotly-based visualization tool for electronic structure data, providing modern interactive band structure and DOS plots in web browsers.

**Scientific domain**: Electronic structure visualization, interactive plotting
**Target user community**: Researchers wanting interactive web-based visualization

## Capabilities (CRITICAL)
- **Interactive Plots**: Dash/Plotly-based visualization
- **Band Structure**: Interactive band structure plots
- **DOS**: Interactive density of states
- **Web-Based**: Browser-based visualization

## Key Strengths
- Modern interactive visualization
- Dash/Plotly framework
- Web browser interface
- Zoom, pan, hover capabilities

## Inputs & Outputs
- **Input formats**: Various DFT output formats
- **Output data types**: Interactive web plots
