# masci-tools

## Overview
**masci-tools** is a Python package providing tools for pre- and post-processing of FLEUR and KKR (Korringa-Kohn-Rostoker) calculations. It is developed by the Materials Science group at Forschungszentrum Jülich.

## Official Resources
- **GitHub**: https://github.com/JuDFTteam/masci-tools
- **Documentation**: https://masci-tools.readthedocs.io/
- **PyPI**: https://pypi.org/project/masci-tools/

## Capabilities
- **FLEUR Support**: Full-potential LAPW post-processing
- **KKR Support**: Green's function method analysis
- **Band Structure**: Electronic band plotting
- **DOS**: Density of states visualization
- **Magnetic Properties**: Spin-resolved analysis

## Key Features
- FLEUR inp.xml parsing and generation
- KKR output analysis
- Integration with AiiDA workflows
- Visualization utilities
- Schema-based XML handling

## Inputs & Outputs
- **Inputs**: FLEUR (inp.xml, out.xml), KKR output files
- **Outputs**: Plots, processed data

## Installation
```bash
pip install masci-tools
```
