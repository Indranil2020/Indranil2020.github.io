# aiida-fleur

## Overview
**aiida-fleur** is an AiiDA plugin for running FLEUR (Full-potential Linearized Augmented Plane Wave) calculations. It provides workflow automation for all-electron DFT calculations.

## Official Resources
- **GitHub**: https://github.com/JuDFTteam/aiida-fleur
- **Documentation**: https://aiida-fleur.readthedocs.io/

## Capabilities
- **FLEUR Calculations**: FP-LAPW method
- **Workflow Automation**: AiiDA integration
- **Band Structure**: Electronic structure workflows
- **Magnetic Properties**: Spin-polarized calculations

## Key Features
- AiiDA workflow management
- FLEUR code interface
- High-throughput support
- Convergence workflows

## Requirements
- AiiDA framework
- FLEUR code

## Installation
```bash
pip install aiida-fleur
```
