## Overview
aiida-fleur is an AiiDA plugin for running FLEUR (Full-potential Linearized Augmented Plane Wave) calculations, providing workflow automation for all-electron DFT calculations.

**Scientific domain**: FLEUR FP-LAPW, workflow automation
**Target user community**: FLEUR users needing automated workflows

## Capabilities (CRITICAL)
- **FLEUR Calculations**: FP-LAPW method
- **Workflow Automation**: AiiDA integration
- **Band Structure**: Electronic structure workflows
- **Magnetic Properties**: Spin-polarized calculations

## Key Strengths
- AiiDA workflow management
- FLEUR code interface
- High-throughput support
- Convergence workflows

## Requirements
- AiiDA framework
- FLEUR code
