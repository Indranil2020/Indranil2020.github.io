# QEPlotter

## Overview
**QEPlotter** is a Quantum ESPRESSO band structure, DOS/PDOS, and fatband plotting toolkit with automated band gap detection and bilayer stacking analysis. It provides both command-line and GUI interfaces.

## Official Resources
- **GitHub**: https://github.com/topics/quantum-espresso (search for QEPlotter)
- **Compatible with**: Quantum ESPRESSO 5.4.0+

## Capabilities
- **Band Structure**: Electronic band plotting with high-symmetry paths
- **DOS/PDOS**: Density of states visualization
- **Fatbands**: Orbital-projected band structures
- **Band Gap Detection**: Automated direct/indirect gap identification
- **Bilayer Analysis**: Stacking analysis for 2D materials

## Key Features
- GUI interface for easy use
- Automated band gap detection
- Support for spin-polarized calculations
- Publication-ready figures
- Quantum ESPRESSO native support

## Inputs & Outputs
- **Inputs**: Quantum ESPRESSO output files (bands.dat, dos.dat, pdos files)
- **Outputs**: Matplotlib figures, data files
