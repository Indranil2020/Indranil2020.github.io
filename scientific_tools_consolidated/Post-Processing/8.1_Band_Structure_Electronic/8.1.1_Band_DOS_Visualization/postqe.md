# postqe

## Overview
**postqe** is a Python package for post-processing calculations from Quantum ESPRESSO. It provides functions to analyze and visualize results including charge density, potentials, and electronic structure.

## Official Resources
- **GitHub**: https://github.com/QEF/postqe
- **Documentation**: Available in repository

## Capabilities
- **Charge Density**: Plot on 1D/2D sections
- **Potentials**: Bare/Hartree/total potential visualization
- **EOS Fitting**: Equation of state analysis
- **Band Structure**: Electronic band plotting
- **DOS**: Density of states analysis

## Key Features
- Native Quantum ESPRESSO support
- 1D and 2D plotting capabilities
- Equation of state fitting (Murnaghan, Birch-Murnaghan)
- Python API for scripting
- Integration with ASE

## Inputs & Outputs
- **Inputs**: Quantum ESPRESSO output files (xml, dat files)
- **Outputs**: Matplotlib figures, numerical data

## Installation
```bash
pip install postqe
```
