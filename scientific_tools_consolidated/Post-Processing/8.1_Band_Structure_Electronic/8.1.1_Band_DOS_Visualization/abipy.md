# AbiPy

## Overview
**AbiPy** is a comprehensive Python package for analyzing ABINIT output files and automating workflows. It provides tools for post-processing electronic structure, phonon, and many-body perturbation theory calculations.

## Official Resources
- **GitHub**: https://github.com/abinit/abipy
- **Documentation**: https://abinit.github.io/abipy/
- **PyPI**: https://pypi.org/project/abipy/

## Capabilities
- **Band Structure**: Electronic band plotting and analysis
- **DOS/PDOS**: Density of states visualization
- **Phonons**: Phonon band structures and DOS
- **GW/BSE**: Many-body perturbation theory results
- **Wannier90**: Interface with Wannier interpolation
- **Workflows**: AbiPy flows for automated calculations

## Key Features
- Comprehensive ABINIT support
- Command-line tools (abiview.py, abiopen.py)
- Jupyter notebook integration
- Publication-quality figures
- Database integration

## Inputs & Outputs
- **Inputs**: ABINIT NetCDF files (GSR.nc, PHBST.nc, DDB, etc.)
- **Outputs**: Matplotlib figures, data exports
