# py4vasp

## Overview
**py4vasp** is the official Python interface for VASP, developed by the VASP team. It provides a modern, user-friendly way to extract and visualize data from VASP calculations, optimized for Jupyter environments.

## Official Resources
- **GitHub**: https://github.com/vasp-dev/py4vasp
- **Documentation**: https://www.vasp.at/py4vasp/latest/
- **VASP Wiki**: https://www.vasp.at/wiki/

## Capabilities
- **Band Structure**: Plot electronic band structures
- **DOS/PDOS**: Density of states visualization
- **Charge Density**: Electron density analysis
- **Magnetization**: Magnetic moment visualization
- **Forces & Stress**: Structural analysis
- **Phonons**: Vibrational properties (VASP 6+)

## Key Features
- Official VASP support
- Jupyter notebook integration
- Interactive plotting with plotly
- Export to common formats (CSV, JSON)
- Modern replacement for p4vasp

## Inputs & Outputs
- **Inputs**: VASP HDF5 output files (vaspout.h5), traditional VASP files
- **Outputs**: Interactive plots, data exports

## Installation
```bash
pip install py4vasp
```
