# bands4vasp

## Overview
**bands4vasp** is a post-processing package for the analysis of unfolded eigenstates in VASP. It provides tools for band structures, 2D and 3D Fermi surfaces, Fermi vectors, and spectral functions.

## Official Resources
- **GitHub**: https://github.com/QuantumMaterialsModelling/bands4vasp
- **Publication**: J. Phys. Chem. C 2021, 125, 21, 11714-11724

## Capabilities
- **Unfolded Bands**: Supercell band structure unfolding
- **Fermi Surfaces**: 2D and 3D Fermi surface visualization
- **Spectral Functions**: Energy-resolved spectral analysis
- **Fermi Vectors**: k-space Fermi vector extraction

## Key Features
- VASP 6.2.1+ support
- Unfolding patch integration
- High-quality visualization
- Spectral function analysis
- Publication-ready output

## Requirements
- VASP 6.2.1 with unfolding patch
- Python 3.x

## Inputs & Outputs
- **Inputs**: VASP output files with unfolding data
- **Outputs**: Band structures, Fermi surfaces, spectral functions
