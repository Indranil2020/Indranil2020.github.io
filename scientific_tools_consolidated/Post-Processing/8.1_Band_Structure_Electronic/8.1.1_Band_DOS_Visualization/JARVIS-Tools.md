# JARVIS-Tools

## Overview
**JARVIS-Tools** (Joint Automated Repository for Various Integrated Simulations) is an open-source Python package for atomistic data-driven materials design. It provides comprehensive tools for setting up calculations, analysis, and visualization.

## Official Resources
- **Website**: https://pages.nist.gov/jarvis/
- **GitHub**: https://github.com/usnistgov/jarvis
- **Documentation**: https://jarvis-tools.readthedocs.io/
- **PyPI**: https://pypi.org/project/jarvis-tools/

## Capabilities
- **Workflow Automation**: VASP, QE, Wien2k, WTBH, Wannier90, LAMMPS
- **Band Structure**: Electronic band analysis
- **DOS**: Density of states processing
- **Database Access**: JARVIS-DFT, JARVIS-FF databases
- **Machine Learning**: ML model integration

## Key Features
- Multi-code support (VASP, QE, Wien2k, etc.)
- High-throughput workflows
- Database integration
- ML-ready featurization
- Web interface (jarvis.nist.gov)

## Supported Codes
- VASP
- Quantum ESPRESSO
- Wien2k
- BoltzTraP
- Wannier90
- LAMMPS
- GPAW
