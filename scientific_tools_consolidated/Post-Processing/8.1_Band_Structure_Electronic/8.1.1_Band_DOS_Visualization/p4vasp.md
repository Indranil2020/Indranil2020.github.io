# p4vasp

## Overview
**p4vasp** is a classic visualization tool for VASP calculations. It provides a graphical user interface for plotting band structures, density of states, and other electronic structure properties from VASP output files.

## Official Resources
- **Website**: http://www.p4vasp.at/
- **SourceForge**: https://sourceforge.net/projects/p4vasp/

## Capabilities
- **Band Structure**: Electronic band structure plotting
- **DOS/PDOS**: Total and projected density of states
- **Structure Visualization**: Crystal structure display
- **Charge Density**: Electron density visualization
- **Local DOS**: Site-projected electronic structure

## Key Features
- Graphical user interface
- Direct VASP file parsing
- Interactive visualization
- Export to various image formats

## Inputs & Outputs
- **Inputs**: VASP output files (vasprun.xml, EIGENVAL, DOSCAR, PROCAR)
- **Outputs**: Plots, images

## Note
p4vasp is considered legacy software. For new projects, **py4vasp** is recommended as the modern replacement.
