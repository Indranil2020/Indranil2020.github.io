## Overview
AICON2 is a program aimed at fast estimation of transport properties, including electronic transport quantities and thermal transport quantities, using approximate models designed for efficiency. It is positioned for applications where rapid screening is needed.

**Scientific domain**: Thermoelectrics, electronic transport, lattice thermal conductivity (model-based)  
**Target user community**: Researchers performing rapid transport-property estimates and screening

## Theoretical Methods
- Model-based evaluation of transport coefficients (see repository documentation)

## Capabilities (CRITICAL)
- Electronic transport property estimation (workflow-dependent)
- Thermal conductivity estimation (workflow-dependent)
- Designed for fast, screening-oriented usage

## Inputs & Outputs
- **Input formats**: Materials/structure and derived quantities as defined by the AICON2 workflow
- **Output data types**: Tabulated transport properties for analysis

## Interfaces & Ecosystem
- Intended to be used alongside first-principles workflows as a fast estimator (see documentation)

## Limitations & Known Constraints
- Model-based results depend on assumptions/parameters; validate against higher-fidelity methods when needed.
