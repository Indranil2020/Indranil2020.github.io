## Overview
mDCThermalC is a software tool for fast estimation of lattice thermal conductivity using a modified Debye–Callaway type approach. It targets rapid screening and approximate thermal transport evaluation.

**Scientific domain**: Lattice thermal conductivity, thermoelectrics screening  
**Target user community**: Researchers needing fast, model-based lattice thermal conductivity estimates

## Theoretical Methods
- Modified Debye–Callaway model family (as described by the project)

## Capabilities (CRITICAL)
- Rapid estimation of lattice thermal conductivity
- Parameter/model-driven evaluation suitable for screening workflows

## Inputs & Outputs
- **Input formats**: Material parameters required by the model (see repository)
- **Output data types**: Estimated thermal conductivity values and intermediate parameters

## Interfaces & Ecosystem
- Designed for lightweight, fast evaluation

## Limitations & Known Constraints
- Model-based; validate where accuracy is critical.
