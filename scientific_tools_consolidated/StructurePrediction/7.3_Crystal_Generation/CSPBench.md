# CSPBench

## Overview
CSPBench is a benchmark suite for evaluating crystal structure prediction algorithms. It provides standardized datasets, metrics, and evaluation protocols for comparing CSP methods.

## Theoretical Basis
- Benchmark dataset curation
- Performance metrics (match rate, RMSD, Chamfer distance)
- Standardized evaluation
- Algorithm comparison
- Reproducible benchmarking

## Key Capabilities
- 180 benchmark crystal structures
- Multiple evaluation metrics
- Standardized protocols
- Algorithm comparison framework
- Open-source codebase

**Sources**: arXiv:2407.00733

## Inputs & Outputs
- **Input formats**: Predicted structures, reference structures

## Interfaces & Ecosystem
- **Algorithms**: USPEX, CALYPSO, DiffCSP, etc.
- **Metrics**: Match rate, RMSD, Chamfer distance
- **Datasets**: 180 benchmark structures

## Workflow and Usage
1. Run CSP algorithm on benchmark set
2. Collect predicted structures
3. Evaluate with CSPBench metrics
4. Compare with other algorithms
5. Report standardized results
