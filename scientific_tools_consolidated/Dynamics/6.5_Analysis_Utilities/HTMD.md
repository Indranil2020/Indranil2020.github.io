## Overview
HTMD (High-Throughput Molecular Dynamics) is a programmable, extensible platform for molecular simulations. It provides tools for system preparation, simulation management, adaptive sampling, and Markov state model analysis.

**Scientific domain**: High-throughput MD, adaptive sampling, Markov state models  
**Target user community**: Researchers running large-scale MD campaigns
