## Overview
MDANSE (Molecular Dynamics Analysis for Neutron Scattering Experiments) is a Python application for computing properties from molecular dynamics trajectories that can be directly compared with neutron scattering experiments.

**Scientific domain**: Neutron scattering analysis, MD trajectory analysis  
**Target user community**: Researchers comparing MD with neutron scattering experiments

## Theoretical Methods
- Dynamic structure factor S(Q,ω)
- Intermediate scattering function
- Density of states
- Mean square displacement
- Velocity autocorrelation
