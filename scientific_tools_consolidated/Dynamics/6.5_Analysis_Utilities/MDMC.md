## Overview
MDMC (Molecular Dynamics Monte Carlo) is a Python package for refining classical molecular dynamics simulations against experimental data, particularly neutron and X-ray scattering measurements. It uses Monte Carlo optimization to adjust force field parameters.

**Scientific domain**: MD refinement, experimental data fitting, scattering analysis  
**Target user community**: Researchers validating MD against experiments

- Force field refinement
- Experimental data fitting
