## Overview
mlmod is a machine learning package for data-driven modeling in molecular dynamics simulations. It provides a LAMMPS interface for using ML models as force fields and enables integration of PyTorch models directly into LAMMPS simulations.

**Scientific domain**: ML-LAMMPS integration, data-driven MD  
**Target user community**: LAMMPS users wanting ML potential integration
