## Overview
MixPI (Mixed-time slicing Path Integral) is a software package for path integral molecular dynamics simulations using mixed-time slicing techniques. It enables efficient simulation of nuclear quantum effects with reduced computational cost compared to standard PIMD.

**Scientific domain**: Path integral MD, nuclear quantum effects, mixed-time slicing  
**Target user community**: Researchers studying quantum nuclear effects efficiently
