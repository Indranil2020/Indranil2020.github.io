## Overview
XTANT-3 (X-ray-induced Thermal And Nonthermal Transitions) is a hybrid code for simulating material response to femtosecond X-ray irradiation. It combines Monte Carlo for high-energy electrons, tight-binding molecular dynamics, and Boltzmann collision integrals for electron-phonon coupling.

**Scientific domain**: X-ray matter interaction, ultrafast dynamics, electron-phonon coupling  
**Target user community**: Researchers studying X-ray induced material modifications
