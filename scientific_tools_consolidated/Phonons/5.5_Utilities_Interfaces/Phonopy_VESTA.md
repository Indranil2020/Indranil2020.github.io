## Overview
Phonopy_VESTA is a tool for exporting phonon eigenvectors from Phonopy calculations to VESTA format for visualization. It enables 3D visualization of atomic displacement patterns using the popular VESTA crystal structure viewer.

**Scientific domain**: Phonon visualization, eigenvector export  
**Target user community**: Researchers visualizing phonon modes in VESTA
